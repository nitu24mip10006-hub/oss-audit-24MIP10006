## Scripts Description

### Script 1: System Identity Report
Displays system information like user, date, and software.

### Script 2: FOSS Package Inspector
Checks if Python is installed and shows version and license.

### Script 3: Disk and Permission Auditor
Displays directory and user-related system details.

### Script 4: Log File Analyzer
Creates a log file and counts error messages.

### Script 5: Open Source Manifesto Generator
Takes user input and generates a personalized manifesto.
